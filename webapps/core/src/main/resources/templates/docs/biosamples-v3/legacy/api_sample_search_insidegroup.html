Sample Search inside a SampleGroup

  • It returns all the samples that are inside the SampleGroup parameter and that also match the query(e.g.  /biosamples/xml/groupsamples/SAMEG23004/query=cancer);
  • It is possible to use paging (  e.g./biosamples/xml/groupsamples/SAMEG23004/query=cancer&page=3&pagesize=10);
  • Always return a result summary information;


Authentication

N/A



Arguments

query (required)
sortby (optional - default value=relevance)
sortorder (optional - default value=descending)
page (optional - default value=10; maximum value=500)
pagesize (optional - default value=1)



Example Request

Get xml/groupsamples/SAMEG23004/query=cancer&page=2&pagesize=20
