Submissions

This webpage provides a service for submitting SampleTab format files as used by the BioSample Database.

Please provide the SampleTab file you wish to submit into the form below. You will then be able to download a version of that file that has been corrected and accessioned.

Please note that due to the number of checks we run, large files (500 samples and over) may require more processing time. This could result in a timeout error being returned in your browser. Should you experience such an issue please or you have any other question about the service, contact us at biosamples@ebi.ac.uk for support attaching the SampleTab file you were trying to submit

If you need an API key please contact biosamples-submissions@ebi.ac.uk
