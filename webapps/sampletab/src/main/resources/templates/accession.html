Accessions

This webpage provides a service for assigning accessions to SampleTab format submissions as used by the BioSample Database.

Please provide the SampleTab file you wish to accession into the form below. You will then be able to download a version of that file that has had sample and group accessions assigned.

Note that the downloaded sampletab will contains only the sample name and accessions with a placeholder description as the intended use of this service is not to submit sample metadata

If you have any questions about this service, please contact biosamples@ebi.ac.uk and attach the SampleTab file you were trying to accession.

If you need an API key please contact biosamples-submissions@ebi.ac.uk
